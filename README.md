# Simple chat based on NodeJS (Express) / Vue 2 / SocketIO

## Featurers
* NodeJS 8 + Express
* Vue 2
* SocketIO
* MondoDB 3 + Mongoose
* Webpack 3
* Bootstrap 4 (beta)
* FontAwesome
* Sass


## Installation

For installing you should run next commands:
* `git clone https://github.com/NMFES/chat.git`
* `cd chat`
* `npm install`
* `sudo npm install nodemon -g`

Also you need to install MongoDB
